# News Website
This is a static web app that helps us know real time news using the NewsAPI.This website is made by using HTML,CSS and Java Script. I have used NewsAPI . This website get the data from News 
API and displays the details, image and also redirect to its main source. News categories are also included like : Sports, Finance and politics etc.
